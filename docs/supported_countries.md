## 참고사항

- 이 목록은 FMP API v3의 `/get-all-countries` 엔드포인트를 통해 조회한 결과입니다.
- 일부 국가의 경우 데이터의 가용성이 제한적일 수 있습니다.
- 각 국가별로 제공되는 금융 데이터의 범위와 깊이는 다를 수 있습니다.
- 실시간 시장 데이터는 주요 금융 시장이 있는 국가들에 대해 우선적으로 제공됩니다.
